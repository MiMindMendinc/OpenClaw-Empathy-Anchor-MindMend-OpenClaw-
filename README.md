
# OpenClaw-Empathy-Anchor-MindMend-OpenClaw

Customized OpenClaw.ai with empathy anchor for ethical, offline AI supporting youth mental health and safety in Michigan. Privacy-first tools for families in Owosso.

## Getting Started
1. Clone the repo: `git clone https://github.com/MiMindMendinc/OpenClaw-Empathy-Anchor-MindMend-OpenClaw.git`
2. Install: `npm install`
3. Run: `npm start` (connect to Telegram/WhatsApp for chatting).
